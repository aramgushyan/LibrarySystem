БИБЛИОТЕЧНАЯ СИСТЕМА

Библиотечная система – учебный проект, разработанный в рамках лабораторной работы.
Приложение построено на чистой архитектуре и представляет собой монолитное приложение.

Используемые технологии: C# / .NET, ASP.NET Core, Entity Framework Core, PostgreSQL.

В отличие от первого проекта библиотечной системы, данный вариант был сосредоточен на проектировании системы.
